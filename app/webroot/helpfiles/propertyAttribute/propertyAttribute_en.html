Property Attribute Help:
  1. Add Property Attribute
    2.
  2. Enter Property Attribute according to language specified..
  3. Select Is Sub Part Flag Yes if property attribute contains the sub part otherwise select No.
  4. Example:
    5.
  5. Survey Number = 119/2/3 Here 119 is main part And /2/3 is a sub part of the Property Attribute.
  6. Click on Save button to save the record.
  7. If you want to clear entered data from text box then use Cancel button.
  8. Edit Property Attribute
    9.
  9. For edit purpose select the record from the grid on clicking the edit button of the correspondent record
  10. The data will be available for edit purpose make the required changes and update it
  11. Delete Property Attribute
    12.
  12. To Delete any of the Property Attribute select the delete button of corresponding record from grid.
  13. It will ask you for surety of deletion if yes then it will be deleted.
