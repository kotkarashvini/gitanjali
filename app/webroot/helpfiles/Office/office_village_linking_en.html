Office Village Linking: Help
  1. Go to Office Jurisdiction in Office Category:
  2. Select Division, Select District, Select Sub Division, Select Tehsil, and Select Circle, Select Corporation, Select Office, Select Village for office village linking.
  3. Enter Village mapping code example = 123456, click on Verify button to verify village mapping code.
  4. Select Office for Office Jurisdiction
  5. To view previously mapped villages, Please select "Select office" from drop-down list
