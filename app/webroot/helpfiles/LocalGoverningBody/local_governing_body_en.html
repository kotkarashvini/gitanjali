Local Governing Body Help
  • Notice : It is compulsory to enter information in the column that is required (*).
  1. Add Local Governing Body
  2. Enter Local Governing Body Name according to language specified
  3. Click on Save button to Save a Local Governing Body
  4. Edit Local Governing Body
  5. For edit purpose select the record from the grid on clicking the edit button of the correspondent record
  6. The data will be available for edit purpose make the required changes and update it
  7. Delete Local Governing Body
  8. For Deleting any of Local Governing Body select the delete button of corresponding record from grid.
  9. It will ask you for surety of deletion if yes then it will be deleted.
  10. Example of Local Governing Body: Municipal corporation,Municipal Council,Gram Panchayat,Nagar panchayat,Nagar parishad
