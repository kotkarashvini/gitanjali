Article And Optional Screen Mapping Help File
  1. Add Article And Optional Screen Mapping:
  2. Select Article
  3. Select Minor list
  4. Click on Save button

  5. Edit Article And Optional Screen Mapping:
  6. For edit purpose select the record from the grid on clicking the edit button of the correspondent record
  7. The data will be available for edit purpose make the required changes and update it

  8. Delete Article And Optional Screen Mapping:
  9. For Deleting any of Article And Optional Screen Mapping select the delete button of corresponding record from grid.
  10. It will ask you for surety of deletion if yes then it will be deleted.
  11. For Ex :- Sale deed article require payment screen as a minor list.
