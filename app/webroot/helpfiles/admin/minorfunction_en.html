Minor Function Help File

  1. Edit Minor Function
  2. For edit purpose select the record from the grid on clicking the edit button of the correspondent record
  3. The data will be available for edit purpose make the required changes and update it

  4. Delete Minor Function
  5. For Deleting any of Minor Function select the delete button of corresponding record from grid.
  6. It will ask you for surety of deletion if yes then it will be deleted.
