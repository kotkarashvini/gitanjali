Unit Help File
  1. Add Unit:
  2. Enter Unit description according to language specified.
  3. Select is Standard Unit yes or no.
  4. *If Standard Unit is Yes then follow below steps
  5. Enter Remark
  6. *If Standard Unit is No then follow below steps:
  7. Select Standard Unit from the list.
  8. Enter Conversion Formula
  9. Enter Remark

  10. Edit Unit:
  11. For edit purpose select the record from the grid on clicking the edit button of the correspondent record
  12. The data will be available for edit purpose make the required changes and update it

  13. Delete Unit:
  14. For Deleting any of Unit select the delete button of corresponding record from grid
  15. It will ask you for surety of deletion if yes then it will be deleted

  16. For Ex.-
    For Rural – Are, Acre, Hectare
    For Urban – Square feet, Square metre
