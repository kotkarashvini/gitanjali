PDE Function Entry

  1. Add PDE Function Entry
  2. To create new PDE Function, go to PDE Function Entry in Document Master.
    3.
  3. Enter Function ID.
  4. Enter PDE Function.
  5. Enter Function Description according to language specified.
  6. Enter Controller details (It’s a PHP Class).
  7. Enter Action. (It’s a PHP Function of Controller class)
  8. Enter Display Order.
  9. Select Display Flag.
  10. Select Include in Status Yes or No.
  11. Select Required for PDE through citizen Yes or No.
  12. Select Required for PDE through SRO Yes or No.
  13. Select Required for manual Registration Yes or No
  14. Select required for eRegistration Yes or No.
  15. Select required for eFiling Yes or No.
  16. Select Required for Leave and License Yes or No.
  17. Select Required for Cidco citizen PDE Yes or No.
  18. Select Is Deleted Yes or No.
  19. Click on Save to save PDE function.

  20. Edit PDE Function Entry
  21. For edit purpose select the record from the grid on clicking the edit button of the correspondent record
  22. The data will be available for edit purpose make the required changes and update it

  23. Delete PDE Function Entry
  24. For Deleting any of PDE Function Entry select the delete button of corresponding record from grid.
  25. It will ask you for surety of deletion if yes then it will be deleted.
