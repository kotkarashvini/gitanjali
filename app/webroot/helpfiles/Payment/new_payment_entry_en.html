Payment Entry Help
  1. New Payment Entry tab
  2. Click on New Payment Entry to enter the details.
  3. New Payment Entry (offline Payment Modes):
  4. •Initially Check details in View Citizen Data tab and enter in following fields
  5. •Select Payment Mode
  1. Stamp paper
  2. 1)Enter Stamp Number
  3. 2)Enter Purchase Date
  4. 3)Enter Amount
  5. 4)Select Account Head- For eg :1)Stamp duty,2)Registration Fee,3)Mutation Fees..etc
  1. Demand Draft(DD)/Cheque
  2. 1)Select the Bank Name
  3. 2)Select the Bank Branch Name
  4. 3)Enter IFSC Code
  5. 3)Enter Payment Date
  6. 3)Enter Amount
  7. 4)Select Account Head- For eg :1)Stamp duty,2)Registration Fee,3)Mutation Fees..etc
  1. E-Registration
  2. 1)Enter Certificate Number
  3. 2)Enter Certificate Date
  4. 3)Enter Amount
  5. 4)Select Account Head- For eg :1)Stamp duty,2)Registration Fee,3)Mutation Fees..etc
  1. Cash
  2. 1)Enter Payment Date
  3. 3)Enter Amount
  4. 4)Select Account Head- For eg :1)Stamp duty,2)Registration Fee,3)Mutation Fees..etc
  1. Bank Challan
  2. 1)Select Bank Name
  3. 2)Enter Challan Date
  4. 3)Enter Challan Amount
  5. 3)Enter Bank Transaction Id
  6. 4)Select Account Head- For eg :1)Stamp duty,2)Registration Fee,3)Mutation Fees..etc
  1. •Click on Save Payment Entry
  2. •Verify New Payment Entry for any selected one is added in Payment tab
