Fee Items
  1. Add Fee Items:
    2.
  2. If Item Type is selected as Input then follow below steps.
  3. Select Is List Item? as Yes or No.
  4. Select Display On Fee Calculation Screen? as Yes Or No.
  5. Select Display On General Informaton Screen? as Yes Or No.
  6. If Item Type is selected as Output then follow below steps.
  7. Select Fee Type.
  8. Select Is Online Payment Compulsory Yes or No.
  9. Enter minimum Value.
  10. Enter maximum value.
  11. Enter account head code.
  12. Select Fee Rounding flag Yes or No.
  13. If rounding flag is selected as Yes then Select Rounding from the list.
  14. Click on Save to save Fee Item.

  15. Edit Fee Items:
    16.
  16. For edit purpose select the record from the grid on clicking the edit button of the correspondent record.
  17. The data will be available for edit purpose make the required changes and update it

  18. Delete Fee Items:
    19.
  19. For deleting any of Fee Item, select the delete button of corresponding record from grid.
  20. It will ask you for surety of deletion if yes then it will be deleted.

  21. For Ex. :- User can add fee rules as per requirement. Rules will be defined for each Fee Type.
    • Major Fee Types/account heads--> Online, Counter etc.
    These fee types/ account heads will be available during fee rule creation and payment of fees at the time of registration.
    • Fee Items--> Market Value, Agreement Value, Area of the flat, Under construction, Stamp Duty, LBT, VAT, Registration Fee etc.
    • Fee Items--> Enter Market/ Sale value, Enter no. of shares, Enter face value of shares
    Fee items are items required while calculating the fees. These items are to be mapped with fee rules. ( Ex. for copying fee No. of pages is the fee item).
